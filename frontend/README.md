# Frontend

## Project setup
```
npm install
```
### create .env File and  set VUE_APP_API_URL Value with your api url 

```
VUE_APP_API_URL : Your_api_url
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
